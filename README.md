# Counter Application

This repository contains a **Counter Application** built with **HTML**, **JavaScript**, and **CSS**. The application allows users to increment, decrement, and reset a counter interactively.

## Language Composition

- **HTML (38.4%)**: Structures the application interface.
- **JavaScript (33.6%)**: Implements the counter functionality and user interactions.
- **CSS (28%)**: Styles the application for a clean and responsive design.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/utasar/counter.git
   cd counter
   ```

2. Open the application:
   - Open the `index.html` file in your browser.

3. Use the app:
   - Interact with the buttons to increment, decrement, or reset the counter.

This project is a simple implementation to demonstrate dynamic functionality with JavaScript and a user-friendly design.
